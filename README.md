# Machine Learning Programming Assignments

This repository contains my solutions to the programming assignments from the **Machine Learning** course taught by **Andrew Ng** (Stanford University / Coursera). The assignments are implemented in MATLAB/Octave and cover a wide range of fundamental machine learning algorithms.

## 📚 Course Information
- **Course:** Machine Learning (Stanford CS229, offered via Coursera)
- **Instructor:** Andrew Ng
- **Platform:** Coursera
- **Language:** MATLAB / Octave

## 📁 Assignments

| Exercise | Topic | Description |
|----------|-------|-------------|
| `machine-learning-ex1` | Linear Regression | Implement linear regression with one and multiple variables, gradient descent, and normal equations. |
| `machine-learning-ex2` | Logistic Regression | Implement logistic regression, regularization, and use it to build a binary classifier. |
| `machine-learning-ex3` | Multi-class Classification & Neural Networks | One-vs-all logistic regression and a pre-trained feedforward neural network for digit recognition. |
| `machine-learning-ex4` | Neural Networks Learning | Backpropagation algorithm to learn parameters for a neural network. |
| `machine-learning-ex5` | Regularized Linear Regression & Bias/Variance | Regularized linear regression, learning curves, polynomial regression, and bias-variance diagnostics. |
| `machine-learning-ex6` | Support Vector Machines | SVM with Gaussian kernels, spam email classification. |
| `machine-learning-ex7` | K-means Clustering & PCA | K-means clustering for image compression, principal component analysis for dimensionality reduction. |
| `machine-learning-ex8` | Anomaly Detection & Recommender Systems | Gaussian anomaly detection, collaborative filtering for movie recommendations. |
